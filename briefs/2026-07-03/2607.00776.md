# From Prediction Uncertainty to Conformalized Distance Fields for Safe Motion Planning

- **arxiv-id**: 2607.00776v1 ／ **topic**: planner_ai ／ **project**: P1 ／ **relevance**: 5/5
- **URL**: http://arxiv.org/abs/2607.00776v1

## 何を解決する論文か
動的環境での安全なモーションプランニングは、予測障害物運動の不確実性をリアルタイム性を犠牲にせず扱う必要がある。既存のconformal手法は障害物ごとの予測誤差を1本のスカラーに集約するため空間整合性を失い、シーン密度に対してスケールしない。本論文は**予測距離場(distance field)全体を一括でconformalize**し、分布フリーかつ場レベルの下界を与えることで「制約を満たす軌道は必ず安全」という一様な安全保証を実現する。

## 手法の核心
- **Functional Conformal Prediction (FCP)**: スカラーでなく予測距離場そのものをconformal化し、field-levelの下界を導出。制御空間のサンプリング方法に依存せず安全が従う。
- **低ランク性の活用**: 残差距離場が経験的に低ランクかつほぼ時間不変 → 係数空間で下界が分解可能になり計算が軽い。
- **オフライン+オンライン2段構え**: functional PCA + Gaussian-mixture inductive conformal でエンベロープをオフライン適合、軽量な adaptive functional conformal (AFCP) で低次元ベクトルのみオンライン補正。分布シフト下でも長期カバレッジを維持。
- **FCP-MPC**: エンベロープを締めた安全制約としてサンプリングベースMPCに埋め込む。1ステップ計算コストが障害物数にほぼ非感受。
- **検証**: ETH–UCY歩行者ベンチ + 最大280動的障害物の3D quadrotorタスクで、点ごと/自己中心conformalベースラインより保守性・コスト面で有利。

## P1(Planner AI + 評価)への適用可能性
- **評価軸としての価値が高い**: 自社AVプランナーの安全性を「分布フリーの被覆保証」で定量化。closed-loop評価に「certified-safe率」「過保守(到達不能/不要減速)率」を新指標として組み込める。
- 予測モジュール(trajectory prediction)の不確実性を下流プランナーの安全制約へ**原理的に接続**する枠組み。予測→計画の継ぎ目を厳密化できる。
- ETH–UCY歩行者ベンチはAV都市シーンに近く、社内ベンチとの比較がしやすい。

### 試すべき実験アイデア
1. 社内予測器の出力にFCPを被せ、既存の固定マージン安全制約 vs FCP制約で**衝突率と過保守(不必要な急減速/停止)率**をclosed-loopで比較。
2. 残差距離場の低ランク性が自社シーン(高速道路 vs 交差点)で成立するか検証 → PCA成分数と計算コストのトレードオフ表を作る。
3. 分布シフト実験: 学習と異なる天候/密度でAFCPオンライン更新の被覆維持を測定し、キャリブレーション劣化の早期検知シグナルに使えるか評価。
