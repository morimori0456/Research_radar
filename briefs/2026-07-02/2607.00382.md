# Vitality-Aware Compression for Efficient Image-to-Shape Diffusion Transformers

- **arxiv-id**: 2607.00382v1 ／ **topic**: fm_distillation ／ **project**: P2 ／ **relevance**: 3/5
- **URL**: http://arxiv.org/abs/2607.00382v1

## 何を解決する論文か
image-to-shape の Diffusion Transformer (DiT) は3D形状生成で高性能だが大型でリソース制約下では計算的に禁止的。既存の拡散モデル圧縮戦略は他ドメイン向けで3D生成へ直接転用しにくく、既存の3D効率化は推論速度中心でバックボーン圧縮を扱わない。本論文は image-to-shape DiT に特化した**幾何認識の圧縮フレームワーク**を提案し、形状忠実度を保ったままモデルサイズを大幅削減する。

## 手法の核心
- **層の非一様重要度の観察**: 3D DiTの層は形状合成への重要度が層ごとに大きく異なる → 一律圧縮でなく重要度に応じた圧縮。
- **vitality-guided フレームワーク**: (1) structured pruning、(2) adaptive quantization、(3) targeted fine-tuning を統合。重要度(vitality)で圧縮強度を配分。
- **成果**: SOTAの image-to-3D モデル群で最大**66%のモデルサイズ削減**、忠実度はフルサイズと同等。
- **plug-and-play**: 多様なモデルに適用可能な汎用ソリューションとして位置づけ。

## P2(Foundation Model 蒸留)への適用可能性
- 純粋な蒸留ではないが、**「層ごとの重要度に応じて圧縮強度を変える」という原則**は蒸留/圧縮パイプラインに直接効く。自社の蒸留で全層一律に生徒容量を割り当てている場合の改善余地を示唆。
- pruning + 量子化 + FT を1本のrecipeに束ねている点が実務的。蒸留後の生徒をさらに圧縮する後段としても使える。
- 適用ドメインは3D生成でP2の主戦場と完全一致ではない → relevance 3。着想の移植価値で採用。

### 試すべき実験アイデア
1. 自社モデルで**層ごとのvitality(重要度)指標を測定**し、一律圧縮 vs vitality配分圧縮で同一サイズ予算下の精度を比較。
2. 蒸留→vitality圧縮の2段パイプラインを試し、蒸留単独 vs 蒸留+圧縮で(サイズ, レイテンシ, 精度)のパレートを描く。
3. adaptive quantizationのビット配分を層vitalityに連動させ、INT8一律との差分を測る。
