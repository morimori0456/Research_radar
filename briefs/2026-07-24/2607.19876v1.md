# KineBench: Benchmarking Embodied World Models via IDM-Free Kinematic Grounding

**topic:** next_arch (P3) ・ **arXiv:** 2607.19876v1 ・ **published:** 2026-07-22 ・ hf_upvotes: 0

## 何を解決する論文か
embodied world model (EWM; ロボットの行動に応じた未来映像を生成する世界モデル) の「物理的整合性」をどう評価するかは未解決。closed-loop 評価(生成映像から行動を取り出しシミュで再実行して妥当性を測る)は open-loop より忠実だが、既存手法はほぼ全て **IDM (Inverse Dynamics Model; 映像から「その間に取られた行動」を逆推定するモデル)** に依存する。IDM は訓練分布外の新物体・新シーンで脆く、失敗すると「world model が悪いのか IDM が悪いのか」区別がつかない **attribution ambiguity(帰属の曖昧さ)** が生じる。KineBench はこの曖昧さを **IDM を使わない** closed-loop 評価で減らす。

## 手法の核心
- **明示的な kinematic grounding:** 生成映像に対し、cascaded な visual foundation model(汎用視覚基盤モデルを段列)で各フレームから **6D end-effector pose**(手先の3次元位置+姿勢)を直接抽出し、physics simulator で closed-loop 実行して検証。IDM の学習依存を排除。
- **ロボット中心の 3D 指標を併用:** タスク成功だけでなく、古典的な **SPARC (Spectral Arc Length; 軌道の滑らかさを周波数領域で測る指標)** と **Maruyama Manipulability Index(手先の可操作性)** で、軌道の smoothness と kinematic feasibility を定量化。
- **4段階の評価スイート:** ManiSkill3 上の 20 の manipulation タスクで、basic execution → task transfer → visual OOD(訓練分布外の見た目)generalization → complexity-conditioned scaling を段階的に評価。
- **知見:** frontier モデル群の評価から、embodied video 生成には **タスク複雑度で頭打ちになる非線形スケーリング**が存在し、今後の data-scaling 戦略への実証的指針を与える。

## 追っているプロジェクトへの適用可能性
- **P3(本命):** driving/embodied world model を作るなら「生成の物理的妥当性を IDM 抜きで測る」枠組みは評価の土台になる。特に visual OOD スイートは、我々の world model が新シーンで破綻するかを切り分けられる。
- **P1(横断):** 「planner を closed-loop で評価する際、抽出器(IDM)の誤差と本体の誤差を分離する」という発想は、我々の NAVSIM 系評価にそのまま効く問題意識。手先 pose の代わりに車両状態を foundation model で直接推定 → 再シミュ、という置換で closed-loop 評価の帰属曖昧さを減らせる。

### 試すべき実験
1. 我々の world model rollout に対し、**行動抽出を「学習した IDM」と「foundation model による直接状態推定」の2系統**で行い、closed-loop 成功率の乖離を計測 → attribution ambiguity の大きさを実測。
2. driving 版 SPARC(自車軌道の滑らかさ)+ feasibility 指標を導入し、タスク成功だけに頼らない多軸評価に。
3. visual OOD スイートを模して、訓練外の天候/シーンで world model の物理妥当性が段階的にどこで崩れるかの scaling を測る。
