# WALA: Learning Executable Latent Actions from Action-Labeled Demonstrations and Action-Free Videos

**arXiv:** 2607.11397v1 | **topic:** next_arch (P3) | **score:** 4/5

## 何を解決する論文か

汎用ロボットポリシーは action ラベル付きのデモ(各時刻の制御信号が付いた軌跡)に依存するが、これは収集が高価でスケールしない。一方、人間・ロボットの大量の動画は豊富な物理的インタラクションを含むが、実行可能な action ラベルを欠く。WALA は、ラベル付きデモと **ラベルなし動画の両方から実行可能な latent action (潜在的な行動表現) を学習する** フレームワークで、ラベルの乏しさをラベルなし動画で補ってスケールさせる。

## 手法の核心

- 動画から semantic-geometric な latent action model を事前学習: 現在観測と疎にサンプルした未来観測の間の変化をモデル化する
- 生ピクセルを再構成せず、**DINOv3 特徴空間の delta と dense depth の delta を予測**。これでタスク関連の意味・幾何構造を保ちつつ、見た目の細部への過敏さを減らす
- ポリシー学習時: 事前学習した encoder が安定した latent action ターゲットを供給し、decoder は学習可能な latent world model として働く
- VLA バックボーンが生成する latent action を、(1) ロボット action 予測、(2) latent action ターゲット一致、(3) 未来ダイナミクス予測、の3つで同時に教師づけ。ラベル付きデモは実行可能な制御教師を、ラベルなし動画は action 注釈なしでダイナミクス教師を供給
- RoboTwin で強い性能、RoboCasa で SOTA (平均成功率 75.2%)、実機操作でも汎化改善

## P3 への適用可能性・実験アイデア

- P3 の world model / VLA 構造に直結。特に「**ラベルなし動画からダイナミクスを学ぶ**」構図は、自動運転で豊富なドライブ動画(action = 制御ログが取れないもの含む)を活用したい状況にそのまま写像できる
- 生ピクセル再構成を避け DINOv3 特徴 + depth の delta を予測する設計は、driving world model の予測ターゲット設計への直接の示唆。ピクセル忠実度に引っ張られず意味・幾何を保つのは運転シーン予測で重要
- 実験アイデア: 手元の driving 動画コーパス(action ラベルなし)で latent action model を事前学習し、少量の action ラベル付き運転データで下流ポリシー/軌道予測を fine-tuning。ラベル量を 1%/10%/100% と振って、ラベルなし動画がラベル効率をどれだけ底上げするか測る
- 予測ターゲットの ablation: raw pixel vs DINOv3-delta vs (DINOv3+depth)-delta を driving シーンで比較し、どの表現が下流制御・予測に効くかを確認
