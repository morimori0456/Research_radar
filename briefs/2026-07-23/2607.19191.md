# ABot-World-0: Infinite Interactive World Rollout on a Single Desktop GPU

- **id:** 2607.19191v1 ｜ **topic:** next_arch (P3) ｜ **hf_upvotes:** 148(本日最高)
- **URL:** http://arxiv.org/abs/2607.19191v1
- **authors:** Fan Jiang, Zhaoxu Sun, Mengchao Wang, Ziyu Zhu, Chiyu Wang, ほか

## 何を解決する論文か
action-conditioned video world model(行動で条件付けした動画予測モデル)を、**リアルタイム・長ホライズンの closed-loop 対話** で動かすための総合システム。従来 world model は長時間 autoregressive に回すと drift(誤差の蓄積で世界が崩壊)する、かつ重くてリアルタイム動作しない、という2大課題を、データ基盤・蒸留・推論スタックの三位一体で潰す。単一デスクトップ GPU(RTX 5090)で 720P を最大 16 FPS で streaming する。

## 手法の核心
- **teacher→student 蒸留の二段。** bidirectional(双方向)な action-conditioned teacher を、teacher forcing と **ODE distillation**(拡散/フローの多ステップ生成を少ステップに蒸留する手法)で causal(因果=過去のみ参照)な student に落とす。
- **LongForcing:** student の長い self-rollout を、より長ホライズンの teacher に整合させる訓練。autoregressive drift と distribution shift(自分の生成を入力に戻すと訓練分布から外れる問題)を緩和する — long-horizon 安定化の肝。
- **統一 action interface = 生キーボード入力。** シーン探索と三人称キャラ操作を同じ制御空間で扱う。reference-character memory(参照キャラの外見記憶)で三人称 rollout 中の同一性を保持。
- **データ基盤:** AAA ゲーム・シミュ・ネット動画を横断。WorldExplorer が訓練フィードバック駆動で収集し、14 個の決定論的品質チェック + VLM(Vision-Language Model)評価 + action/text 同期アノテーションで整備。
- **推論スタック co-design:** 軽量 VAE decoder、efficient attention、memory-aware scheduling、low-bit DiT(Diffusion Transformer の低ビット量子化)。action→初フレーム 1.2s、ピーク約 19GiB VRAM。

## P1/P2/P3 への適用可能性と実験アイデア
- **P3(次世代アーキ)本命。** 「long-horizon で崩れない action-conditioned world model」は AD の予測/シミュ中核と直結。特に **LongForcing** と **ODE distillation** は、我々が world model を長距離 rollout で使う際の drift 対策の設計テンプレート。
- **実験:** AD の driving world model に LongForcing 型の「長 self-rollout を長ホライズン teacher に整合」訓練を導入し、30–60s rollout での collision 予測の安定性(drift 曲線)を LongForcing 有無で比較。
- **P2(蒸留)への持ち帰り:** bidirectional teacher → causal student の蒸留 + few-step 化は、重い予測モデルを実車推論予算に落とす recipe そのもの。**実験:** 我々の重い teacher を ODE distillation で 4–8 step の causal student に蒸留し、latency-品質 Pareto を測る。
- **P1(評価):** リアルタイム対話 rollout は、closed-loop 評価環境を軽量に自前構築する足場になりうる(実 sim の代替コスト削減)。
