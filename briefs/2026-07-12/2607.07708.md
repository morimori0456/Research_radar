# SciReasoner: 構造ネイティブな token 語彙で科学の structure-property を推論する foundation model

- **arxiv-id**: 2607.07708 ／ **topic**: sns_wildcard ／ **project**: EXPLORE ／ **relevance**: 2/5 (本枠適合は低い。探索枠として学びの価値で選定) ／ **hf_upvotes**: 83
- **URL**: https://arxiv.org/abs/2607.07708

## 何を解決する論文か
タンパク質・小分子・無機結晶といった科学対象では、機能や反応性が「空間配置・化学組成・周期構造」から立ち上がる (structure-property relationship; 構造が性質を決める関係)。この関係を機械的に説明するには、構造という証拠を科学原理と物理制約 (立体化学・結合・対称性・エネルギー・周期秩序) に照らして解釈する必要がある。SciReasoner は、こうした構造情報を失わずに保持しつつ「どの証拠がどの予測を支えるか」を示せる、multimodal な科学 foundation model を提案する。分野は AD と無関係だが、後述のとおり表現・推論の設計思想に転用余地がある。

## 手法の核心
- **構造の離散化 + 統一語彙 (unified structure-aware vocabulary)**: 座標・topology (原子の結合関係)・周期的な connectivity (結晶の繰り返し接続) をまとめて離散 token 列に変換し、タンパク質・分子・結晶を 1 つの語彙で扱う。異種のモダリティを共通トークン空間に写す設計。
- **structural token を「addressable evidence unit」として扱う**: 各構造 token を、推論中に名指しで参照できる証拠単位として位置づける。予測の根拠が構造のどの部分かを追跡でき、transparency (説明可能性) を担保する狙い。
- **推論トレースの生成**: 単に答えを出すのでなく、化学の retrosynthesis (逆合成; 目的分子をどの前駆体から作るか逆向きに設計する) では fragment 単位の disconnection (結合の切断箇所) と precursor-verification (前駆体の妥当性検証) の過程を明示的に出力する。
- **報告された定量結果**: (1) homology-controlled な Gene Ontology 予測 (配列類似度の影響を統制した機能アノテーション課題) で、低相同・orphan なタンパク質の Cellular Component 予測が F_max 0.42→0.55 に改善。(2) single-step retrosynthesis の精度 0.63→0.72。(3) 材料科学で単体相と化合物相を表現上分離し、high/low band-gap (半導体のバンドギャップの大小) 領域を切り分け。

## 追っているプロジェクトへの適用可能性
本枠 (P1/P2/P3) への直接適用は薄い。価値は「異種の構造化入力を単一 token 語彙に写し、各 token を後段の推論から名指しで参照できる証拠単位にする」という**表現 × 推論の接続様式**にある。P3 の driving VLA/world model では、シーンの構成要素 (周辺 agent・車線・信号・工事区間) をどうトークン化して推論に載せるかが設計の要で、07-11 に読んだ WCog-VLA の agent token や LaMem-VLA の latent memory token と同じ問題圏にある。SciReasoner の「構造を addressable な証拠 token にして推論トレースで根拠を示す」発想は、planner の意思決定を「どの周辺 token を根拠にこう動いたか」と説明可能にする方向 (P1 のインシデント事後検証とも接続) に読み替えられる。ただしドメインが遠く、手法の中核は分子・結晶の離散化なので、深追いは推奨しない。着想メモとして保持するのが妥当。

### 試すべき実験アイデア
1. **scene のトークン化語彙の設計借用**: 周辺 agent・車線・信号を「addressable evidence token」として設計し、planner の出力に「参照した token の id 列」を紐づける小実験。決定根拠のトレーサビリティが上がるか、既存 VLA の attention 可視化と比較する。
2. **推論トレースを教師信号にする発想の移植**: retrosynthesis の「切断箇所 + 検証」トレースに相当するものを運転で定義 (例: 「この agent の予測軌道 → 自車の回避判断 → 制約充足の確認」)。WCog-VLA の Game-CoT データ生成と組み合わせ、根拠付き推論を明示教示できるか検討する。
