# Grasp, Handover, Rotate: Bimanual Object Reorientation via Compositional Diffusion and Energy-Based Optimization（BiCompoDiff）

**topic:** planner_ai (P1) ・ **arXiv:** 2607.21341v1 ・ **published:** 2026-07-23 ・ hf_upvotes: 0
**（注記）** 対象は両腕ロボットの物体再配置で自動運転そのものではない。**「多目的の制約下で軌道を最適化する planner 手法」**として P1 に転用価値があると判断し採用(在庫制約下の 1 枠)。

## 何を解決する論文か
両腕での物体 reorientation(掴む→腕から腕へ渡す→目標姿勢で置く)は、衝突・kinematics(関節の可動制約)・最終姿勢の悪さで「初手の掴みから直接置く」のが無理な時に要る。だが **衝突回避・軌道の滑らかさ・受け渡し可能性・持ち替え安全性**が同時に競合し難しい。BiCompoDiff は、これらを **composable な energy(各制約を「エネルギー(低いほど良いスコア)」として足し合わせ、勾配で同時に満たす)**として扱い、grasp 選択から motion planning までを一括最適化する。

## 手法の核心
- **compositional diffusion + EBM:** 事前学習済みの **grasp diffusion model(掴み姿勢を拡散生成するモデル)**に、**EBM (Energy-Based Model; 状態の良さをスカラーのエネルギーで表し、低エネルギーへ最適化するモデル)**の勾配ガイダンスを注入。複数制約を1つの合成エネルギー地形として重ねる。
- **reverse diffusion 中の gradient guidance:** 拡散の逆過程(ノイズ→サンプル生成)の各ステップで、**衝突回避・軌道 smoothness(differentiable IK = 微分可能な逆運動学で実現)・handover feasibility・regrasp safety**の勾配を足し込み、生成しながら制約を効かせる。
- **Annealed MCMC で精緻化:** 合成エネルギー上で **annealed MCMC(温度を徐々に下げるサンプリングで良い解に収束させる手法)**を掛け、grasp 姿勢をさらに洗練。
- **成果:** シミュの家庭内 reorientation で、強力な sampling ベース baseline 比 **成功率 +20% 超・軌道 smoothness 最大 +37%**(関節変位で測定)。実機でも sim-to-real 転移を確認。

## 追っているプロジェクトへの適用可能性
- **P1(本命・転用):** AD planner でも「衝突回避・快適性(滑らかさ)・実行可能性・安全マージン」が競合する多目的最適化そのもの。**各制約を微分可能エネルギーとして拡散生成の逆過程に勾配注入**する枠組みは、trajectory planner を「事前学習した軌道 prior + 制約エネルギー」で構成する設計に転用しうる。
- **P1(評価):** 制約ごとにエネルギー値が出るので、**「どの制約がどれだけ効いた/破れたか」を軌道単位で分解**でき、planner 評価の解釈性に効く。
- **P3(横断):** driving world model や diffusion planner に対し、生成時ガイダンスで安全制約を後付けする作法の参考。

### 試すべき実験
1. 手持ちの軌道 diffusion prior に、**衝突・comfort(加加速度)・TTC(Time-to-Collision; 衝突までの残り時間)下限**を微分可能エネルギーとして与え、reverse diffusion 中に勾配注入。無ガイダンス比で violation 率と smoothness を比較。
2. **annealed MCMC の精緻化ステップ**を planner 出力に後段付与し、追加コスト vs 制約充足の改善を測る。
3. 制約エネルギーの重み配分を振り、**成功率と smoothness の Pareto**を描いて、多目的トレードオフの制御性を確認。
