# Decision Journal — 意思決定の記録と答え合わせ

> 判断力は測定しないと向上しない。重要な決定を「予測付き」で記録し、
> 四半期レビューが過去の予測と実際を突き合わせる (calibration)。
> Bezos の Regret Minimization と Type 1/2 分類をテンプレートに組み込んである。

## いつ書くか

以下のどれかに該当する決定をしたとき (月2-3件が目安。全部は書かない):
- キャリアに影響する (転職検討、プロジェクトの引き受け/断り、論文ターゲット選定)
- 数ヶ月単位のリソース配分を変える (技術選定、実験の方向転換)
- 「後で振り返りたい」と直感したもの

## 書き方

`_template.md` をコピーして `YYYY-MM-DD_短い名前.md` で保存。5分で書く。
完璧に書こうとしない — 予測と確信度が入っていれば十分。

## 答え合わせ

四半期レビューが「見直し日」を過ぎたエントリを拾い、
予測と実際のズレ・確信度の傾向 (過信/過小評価) を報告する。
